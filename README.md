# 📊 HR Analytics Dashboard | PostgreSQL • Excel • Power BI

## 📌 Project Overview
This project is an end-to-end HR Analytics solution developed using **PostgreSQL, Microsoft Excel, and Power BI**. The objective is to transform raw HR data into meaningful business insights that support strategic HR decision-making.

The project demonstrates the complete analytics workflow, including data cleaning, database design, SQL analysis, master data creation, and interactive dashboard development.

---
## 🎯 Business Problem

Organizations generate HR data from multiple sources, making it difficult to monitor employee performance, attendance, payroll, departmental distribution, and workforce trends.

This project consolidates HR data into a single analytical model and presents key workforce metrics through an interactive Power BI dashboard.

---

# 💡 Key Insights

- Workforce consists of **1,000 employees** across **5 departments** and **4 cities**.
- IT is the largest department with **336 employees (33.6%)**.
- Delhi has the highest employee count (**265 employees**).
- Total monthly payroll is **₹73.34 Million**.
- Sales Executives receive the highest average salary (**₹75,247**).
- Average employee performance rating is **3.73 / 5**.
- Attendance rate is **88%**, indicating strong workforce attendance.
- Employees work an average of **7.68 hours per day**.
- Marketing department records the highest average salary hike (**7.27%**).
